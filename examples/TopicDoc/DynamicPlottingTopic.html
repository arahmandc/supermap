Dynamic plotting

Introduction

SuperMap iClient3D for WebGL provides a 3D dynamic plotting module that allows users to publish dynamic plotting services on the server side, and implements dynamic plotting and situational derivation functions on the client side by accessing the plotting service published by the server. Flexibly plot 3D point labels, line surface labels, and perform situational deductions in 3D scenes.

    • Technical features: Dynamic plotting technology can dynamically plot point labels and line labels, and can independently modify the style, position and shape of each label, and provide 7 kinds of animations to realize 3D situational derivation.
    • Application scenario: It is mainly used to realize three-dimensional dynamic plotting of command and control schemes such as military operations plan, emergency disposal plan, and public security round-up plan in three-dimensional scenes. Through the provision of various types of graphics and model labels with special meanings, it is possible to display the event handling plan very intuitively and accurately on the 3D scene, and improve the efficiency of meeting and communication during event handling.

Introduction

First we have to do the following preparations:

    • Ensure that the server has available plotting services, or we can go to the server service management page and quickly publish a plotting service by associating an available tag library.
    • Add a viewer to the cesiumContainer, and on this basis, access the 3D services (such as terrain data, oblique photographic data, fine building models, etc.) published by SuperMap iServer for dynamic plotting in the corresponding 3D scene.

First, the dynamic plotting topic - mouse plotting

1, create a dynamic plot total control object

First, you need to create a dynamic plot master object and specify the plot service serverUrl and scene scene for it. 

                    var plotting = cesium.Plotting.getInstance({
                                serverUrl : serverUrl,
                                scene : scene
                            });

2, Create a plot layer

Create a plot layer object and add the plot layer to the scene.

                   var plottingLayer = new cesium.PlottingLayer(scene,{name: "plottingLayer"});
                   scene.layers.add(plottingLayer);

3, create a label library manager object

Second, we need to get the label library manager object from the dynamic plot master object.

                    var symbolLibManager = plotting.getSymbolLibManager();

4, create edit controls and draw controls

Finally, you need to create edit controls and draw controls to start mouse plotting and editing.

Create an edit control

                    plotEditControl = new cesium.PlotEditControl(scene,plottingLayer);
                    plotEditControl.activate();

Create a draw control

                    plotDrawControl = new cesium.PlotDrawControl(scene,plottingLayer);

So far, the dynamic plotting initialization work has been completed, and the mouse can be plotted on the client side. In addition, in order to make it easier for users to plot and edit labels, we provide two panels for plotting and properties in examples.

Second, the dynamic plotting topic - programming plotting

SuperMap iClient3D for WebGL dynamic plotting technology supports programming to achieve dynamic plotting. Programming plotting also needs to first create dynamic plotting master control, plotting layer, label library manager object, editing control and drawing control. The implementation method is the same as the first, second, third, and "dynamic plotting-mouse plotting". 4 parts, not repeated here.

1, Create a label

According to the label library encoding, label encoding, position point array can create labels, the statement is as follows:

                   plottingLayer.createSymbol(libID, code, points, function (even) {
                                var geo = even.feature;
                        });
                   // Among them, libID is the code library code, code is the label code, points is the position point array, and function (even){} is the callback function.

2, modify the label properties

The attributes that can be modified by the label are stored in two parts. The attributes unique to the point label are obtained from geo.symbol, and the attributes common to the point label and the line label are obtained from geo.symbolStyle. The following code takes a small number of attributes as an example. 3. The label attribute list specifically lists the main attributes of the label.

                    geo.scale = new cesium.Cartesian3(1, 1.0, 1.0);  //scaling ratio
                    geo. gridLineWidth = 6;  //Bulletin board mode line width
                    geo.gridSurroundLineWidth = 4;  //Bulletin board mode serif width
                    geo.symbolStyle.surroundLineType = 1;  //Serif type
                    geo.symbolRank = 1;  //Label level
                    geo.lineSymbolID = 1;  //Point label edge type

2、动画管理类

SuperMap iClient3D for WebGL动态标绘技术中态势推演主要是利用GOAnimationManager类（动画管理类）完成态势推演功能。为方便用户顺利实现态势推演功能，下面列出GOAnimationManager类使用方法。

(1)获取动画管理对象

                   animationManager = plotting.getGOAnimationManager();

(2)创建动画

                   animationManager. createAnimation(animationType, name, goFeature);
                   // animationType表示动画类型，name为动画名称，goFeature为动画对象
                   // 注意：在创建动画时，必须选中想要创建动画的对象

(3)执行动画

                   animationManager.execute();

(4)播放动画

                   animationManager.play();
                   // 注意：动画播放时可以单个播放，也可以多个动画一起播放。

(5)暂停动画

                   animationManager. pause ();

(6)停止播放动画

                   animationManager. stop ();

(7)重置动画

                    animationManager. reset();

四、态势图管理

SuperMap iClient3D for WebGL动态标绘技术支持态势图管理，包括保存态势图到服务器、打开态势图文件、删除态势图文件、上传态势图文件等功能。态势图管理的各项功能是通过SitDataManager类来完成的。

首先，我们需要通过下面的语句获取态势图管理对象：

                   sitDataManager = plotting.getSitDataManager();
